## ChatGPT Record機能とは

ChatGPTのmacOSデスクトップアプリに搭載されている録音・要約機能です。会議を録音すると自動で要約を生成してくれます。

詳しい機能説明は公式ヘルプを参照してください。

@[card](https://help.openai.com/en/articles/11487532-chatgpt-record)

ここからは、私が実際に使っている運用Tipsです。

## Tip 1: 「議事メモ」専用プロジェクトで管理する

議事録は後から振り返りたいことが多いですよね。しかし、通常のチャットと同じ場所に並ぶと探しづらくなります。

**おすすめ**: 「議事メモ」という専用プロジェクトを作成し、議事録を集約します。

![ChatGPTのプロジェクト一覧画面で「議事メモ」プロジェクトを指し示している画像](/images/chatgpt-record-meeting-tips/02-project-list.png) _「議事メモ」専用プロジェクト_

専用プロジェクトを使うと、過去の議事録が一覧化されて探しやすくなります。検索もプロジェクト内で完結するので、他のチャットと混ざらず便利です。

## Tip 2: 社内フォーマットに変換する

Record機能が生成する要約は便利ですが、求めるフォーマットとは異なることが多いです。

![Record機能の要約結果画面。Overall Summary、Key Points、Action Itemsなどが表示されている](/images/chatgpt-record-meeting-tips/04-summary-result.png) _Record機能が生成する要約（Overall Summary、Key Points、Action Itemsなど）_

そこで、**チャットでフォーマットを指定して変換を依頼**します。

![ChatGPTにフォーマット変換を依頼しているチャット画面](/images/chatgpt-record-meeting-tips/05-format-request.png) _フォーマットを指定して変換を依頼_

私は以下のようなフォーマットを使っています。

:::details 指定するフォーマットの例

```markdown
# 議事録

## 基本情報

| 項目   | 内容 |
| ------ | ---- |
| 会議名 |      |
| 日時   |      |
| 場所   |      |
| 参加者 |      |
| 記録者 |      |

## 議題

## 議論内容

### 議題1:

**報告事項**: **議論**: **決定事項**:

## アクションアイテム

| 担当者 | タスク | 期限 |
| ------ | ------ | ---- |

## 次回会議
```

:::

フォーマットに沿った議事録に変換してくれます。

:::details 変換後の議事録（例）

**決定事項**:

## Tip 3: Markdown形式で共有する

作成されたCanvasはMarkdown形式でコピー・エクスポートできます。

### 操作手順

1. Canvas上部の「...」メニューをクリック
2. 「Copy as Markdown」を選択
3. Slack/Notion等に貼り付け

![CanvasのメニューからCopy as Markdownを選択している画面](/images/chatgpt-record-meeting-tips/07-markdown-export.png) _Canvasから「Copy as Markdown」を選択_

Markdownでコピーできると、Slackやメールでサッと共有できますし、社内WikiやNotionへの貼り付けも簡単です。わざわざ別ツールで整形する必要がありません。

## 注意点

### 話者分離ができない

Record機能は「誰が話したか」を区別しません。複数人の会議を録音しても、発言者の情報は含まれないため、必要に応じて手動で補足する必要があります。

### 英語で出力されることがある

日本語の会議を録音しても、要約が英語で出力されることがあります。その場合は、チャットで「日本語に翻訳して」と指示すれば変換できます。

## まとめ

ChatGPT Record機能を使った議事録作成のTipsをまとめました。

1. **専用プロジェクトで管理** - 議事録を一箇所に集約して探しやすく
2. **社内フォーマットに変換** - フォーマットを指定するだけで自動変換
3. **Markdown形式で共有** - チームへの共有が手軽

ZoomやTeamsの議事録機能が使えない環境でも、この方法なら議事録作成の負担を大幅に減らせます。ぜひ試してみてください。
