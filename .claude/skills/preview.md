# preview

Zenn記事のプレビューサーバーを起動するスキルです。

## 使い方

```
/preview
/preview 3000
```

## 手順

1. **プレビューサーバーを起動**
   ```bash
   npx zenn-cli preview
   ```
   - ポート指定がある場合: `npx zenn-cli preview --port <ポート番号>`
   - デフォルトポート: 8000

2. **ブラウザで確認するURLを案内**
   ```
   http://localhost:8000
   ```

3. **プレビューの使い方を説明**
   - 記事一覧が表示される
   - 記事をクリックすると実際の見た目でプレビューできる
   - ファイルを保存するとホットリロードされる

4. **終了方法を案内**
   - `Ctrl+C` でサーバーを停止

## 注意事項

- プレビューはローカル環境のみで閲覧可能
- published: false の記事もプレビューできる
- 画像はpublic/配下に配置するか、外部URLを使用
